# アルゴリズム翻訳学習プロジェクト

## 目的

C言語から Pythonへの翻訳を通じた学習。

## 底本

Cで書くアルゴリズム
: 疋田輝雄
: 初版第6刷（2001）
: サイエンス社

プログラミングの宝箱 アルゴリズムとデータ構造
: 紀平拓男 春日伸弥
: 初版（2003)
: ソフトバンクパブリッシング

## 注意事項

- 原語版
    - 原則として底本の記述を忠実に再現する
    - コードフォーマットに関しては多少変更しても良い
    - 処理は改変してはならない
- 直訳版
    - Cの処理の流れをそのままPythonとして直訳する
    - 直訳できない処理については翻訳しても構わないが、極力忠実にする
- 翻訳版
    - Pythonの流儀に習ったコードとして翻訳する
    - Cの手順より良い手段があるなら積極的に改変する
    - テストコードも書く

## 各アルゴリズムのディレクトリ構成

```
algorithm/
┝── 01_gcd/  - アルゴリズム名
│   ┝── original.c   - 原語版（元のC実装）
│   ┝── direct.py    - 直訳版
│   ┝── pythonic.py  - 翻訳版（Pythonic実装）
│   ┝── test.py      - テストコード
│   └── README.md    - 学習メモ、ベンチマーク結果
┝── NN_ALGORITHM_NAME/
...
...
```
